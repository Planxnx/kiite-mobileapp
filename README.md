## Getting started
#### Required
- <a href="https://github.com/Planxnx/kiite-webservice">
  Kiite-Webserive
</a>
First, install React Native, Expo cli and other node modules.

```bash
$ npm install -g expo-cli
```

```bash
$ npm install
```

To start the server:
```bash
$ npm start
```
or
```bash
$ npm run ios
```
must connecting to Kiite-Webserivce
